## 🔴 ALERT: Category L (Legal/Normativa) scored 33.5 — well below 70 threshold

### Diagnosis

**Root cause:** The bot's RAG knowledge base does not contain regulatory/legal content (NOM-251, Distintivo H, food safety norms). When it encounters these queries:

1. **L01 (NOM-251):** Intent classifier completely fails — routes to "Reporte de productos más vendidos" instead of informacional/RAG. This is an intent classification bug where the word "alimentos" (food) triggers the sales report path.
2. **L11 (Distintivo H):** Intent correctly classified as "informacional" but the RAG retrieval returns nothing, and the fallback message ("dime en qué módulo estás") is inappropriate for a regulatory question.

### Recommended Fixes

1. **Intent classifier:** Add training examples for regulatory/legal questions so they route to informacional, not sales reports. Keywords: NOM, norma, ley, regulación, distintivo, COFEPRIS, salubridad.
2. **RAG knowledge base:** Add documents covering common Mexican restaurant regulations (NOM-251, NOM-093, Distintivo H, COFEPRIS requirements).
3. **Fallback handling:** When informacional intent has no RAG results, respond with "No tengo información sobre normativas/regulaciones en mi base de conocimiento. Te recomiendo consultar directamente con COFEPRIS o la Secretaría de Turismo." instead of asking about system modules.
