## Key Learnings
- **Railway service names have dashes not underscores**: `pos-inventory-api`, `pos bot api` (with spaces)
- **Railway redeploys ALL services on push to monorepo** — need to configure Watch Paths/Root Directory per service
- **User rule: ALWAYS ask before push/merge** — added to USER.md
- **AdonisJS migrations don't auto-run on Railway** — must be run manually via SQL or migration command
- **macOS screenshots have Unicode narrow no-break space (U+202F)** in filenames — use `ls | grep` + `cp` workaround
