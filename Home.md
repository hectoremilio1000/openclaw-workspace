# 🏠 Home

Bienvenido al vault. Esto es tu segundo cerebro compartido con tu agente de IA.

## 📂 Estructura

- **[[MEMORY]]** — Memoria a largo plazo del agente
- **memory/** — Notas diarias (raw logs)
- **projects/** — Proyectos (GrowthSuite, etc.)
- **notes/** — Tus notas personales, ideas, lo que quieras
- **scripts/** — Scripts de sync, DB, fixes
- **data/** — JSONs, excels, datos
- **assets/** — Screenshots, imágenes, PDFs
- **archive/** — Análisis viejos, videos HTML
- **skills/** — Skills de OpenClaw (deploy-verify, bot-test, etc.)

## 🔧 Archivos del sistema (no tocar)

- AGENTS.md, SOUL.md, USER.md, TOOLS.md, HEARTBEAT.md — Config del agente
- .obsidian/ — Config de Obsidian
